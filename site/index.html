How It Works

Three steps. Zero configuration.

1

Connect your calendars

OAuth into Google Workspace, Microsoft 365, or any CalDAV server. T-Minus requests read/write access and begins incremental sync immediately.

2

T-Minus builds your temporal graph

Events from every account are deduplicated, enriched, and stored in your personal Durable Object. Relationships, patterns, and commitments emerge from the data automatically.

3

Intelligence emerges automatically

Cognitive load scores, drift alerts, deep work analysis, and commitment tracking activate without configuration. Your AI assistant gains 26 tools to manage it all.

Model Context Protocol for Claude, GPT, and any MCP-compatible AI. Natural language calendar management.

MCP Tool Call Example 
// Ask your AI assistant:
// "Find me a 90-minute focus block tomorrow"

// T-Minus MCP tool invocation:
{
  "tool": "find_deep_work_slots",
  "arguments": {
    "date": "2026-02-16",
    "min_duration_minutes": 90,
    "protect": true
  }
}

// Response:
{
  "slots": [
    {
      "start": "09:00",
      "end": "11:30",
      "quality_score": 0.94,
      "context_switches_before": 0,
      "cognitive_load_impact": -1.2
    }
  ],
  "recommendation": "9:00-11:30 is optimal.
    No prior meetings. Load drops 1.2 points."
}
# REST API
curl -H "Authorization: Bearer $TOKEN" \
  https://api.tminus.dev/v1/intelligence/load \
  -d '{"date": "2026-02-16"}'

# Returns cognitive load score, breakdown,
# deep work hours, and optimization suggestions
